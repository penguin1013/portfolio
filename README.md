# Portfolio Project

This is a React portfolio project built with Vite.

## üöÄ Development Workflow

Follow these steps to modify and deploy your website.

### 1. Local Development (Coding & Testing)
To run the project on your local machine:

1.  **Install dependencies** (first time only):
    ```bash
    npm install
    ```
2.  **Start the local server**:
    ```bash
    npm run dev
    ```
3.  **View in Browser**: Open the URL shown in the terminal (e.g., `http://localhost:5173/portfolio/`).

### 2. Deploying to GitHub Pages (Live Website)
To update the live website seen by others:

1.  **Run the deploy command**:
    ```bash
    npm run deploy
    ```
    This command will automatically build your project and push it to the `gh-pages` branch.
2.  **Visit your site**:
    [https://penguin1013.github.io/portfolio](https://penguin1013.github.io/portfolio)

### 3. Saving Source Code
To save your source code changes to the repository:

```bash
git add .
git commit -m "Update portfolio content"
git push
```
